Webhook Payload Format

  "bucket_name": "{{.Bucket.Name}}",
  "resource_id": "...",
  "resource_url": "https://...",
  "resource": {
    "hash": "sha256...",
    "size": 12345,
    "content_type": "image/png",
    "extension": ".png"
  }
}

Webhooks are sent immediately when resources are created or deleted. Custom headers are included with each request.
